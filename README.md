# Profile

A centralized repository containing all professional profile data, resume templates, and presentation materials for various platforms.

## Overview

This repository serves as a single source of truth for maintaining consistent professional presence across multiple platforms including resumes, LinkedIn, GitHub, and other professional networks.


## Features

- Centralized data management through JSON structure
- Multiple resume template versions
- Platform-specific profile formats (LinkedIn, GitHub)
- Automated PDF generation capabilities
- Version control for all profile materials

## Usage

### Updating Profile Data

Edit the `data/data.json` file to update your information. All templates automatically pull from this central data source.

### Generating Resume

Open any HTML template file in your browser. The template will fetch data from `data.json` and render the resume. Use the download button to generate a PDF version.

### Customizing Templates

Templates are designed to be modular and easy to customize. Modify the HTML/CSS in the template files while keeping the data structure intact.

## Data Structure

The profile data follows a standardized JSON schema including:

- Personal information and contact details
- Professional experience and achievements
- Education and certifications
- Technical skills and competencies
- Projects and portfolio items
- Professional philosophy and values

## Templates

All templates are responsive, print-friendly, and designed for professional presentation. Templates maintain consistent styling while allowing easy customization.

## Maintenance

This repository is regularly updated to reflect current professional status, achievements, and experience. Historical versions are archived for reference.
